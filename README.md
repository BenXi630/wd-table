# wd-table

## Install
```
yarn add wd-table
```

### Quick Start
```
import Vue from 'vue'
import WdTable from 'wd-table'
import "wd-table/dist/wd-table.css"

Vue.use(WdTable)

// or
import {
  TableGroup,
  Table
  // ...
} from 'wd-table'

Vue.component(TableGroup.name, TableGroup)
Vue.component(Table.name, Table)
```
